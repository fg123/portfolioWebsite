Web Design

Modern technologies like HTML5 and CSS3 for UI design and layout organization, Javascript and jQuery for stunning effects and fluid transitions, and Bootstrap for grid and mobile layouts.

Web Backend

PHP MySQL
Using the server-side PHP and MySQL to manage databases allows for user-specific interactive websites. I employ PHP and MySQL for tools like forums, blogs, contact forms and user management systems. PHP and MySQL are both very flexible and easy to integrate for any type of website.

Game Design

C# C++ .NET SFML Unity3D XNA
I also do some game design using Unity and C# as my primary setup. Unity allows me to use many built in game structures so I can focus primarily on designing game play instead of focusing on basic functions like drawing sprites or calculating vectors. I also have experience in less visual game development libraries like SFML, which is based on C++, and XNA, which is a bit more outdated and based on C#.
