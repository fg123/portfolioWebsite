Syntax

struct name => (field1, field2, ...);

Usage

struct creates a predefined structure that allows multiple values to be grouped together.

When a struct is defined with n fields, Wendy creates n + 1 functions: the constructor function, and accessor functions for each of the fields.

struct point => (x, y);
let p = point(10, 20);
point_x(p); // outputs 10
point_y(p); // outputs 20

In the previous example, when struct point => (x, y); is called, the following functions are defined:

//constructor
let point => (x, y) {
	let newobj[2]; // creates new array in memory
	set newobj[0] = x; // assign values based on constructor
	set newobj[1] = y; 
	ret (&newobj); // returns pointer to new instance of structure
}
// accessor function for x
let point_x => (objptr) {
	ret  *(objptr + 0); // returns the first element of the array
}
// accessor function for y
let point_y => (objptr) {
	ret  *(objptr + 1); // returns the second element of the array
}
