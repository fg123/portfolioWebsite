Syntax

if (condition) {
	statement;
}
else if (condition) { // optional, can have any amount of else if
	statement2;
}
else { // optional, but can only have one
	statement3;
}

Usage

if is a control flow modifier that executes the resulting block of code if the given condition is true. Conditions are evaluated in order and once a condtion evaluates to true, the function block is executed and the other conditions are disregarded.
