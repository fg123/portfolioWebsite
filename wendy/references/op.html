Math Operators

+ // add two numbers or concatenates two strings
- // subtracts two numbers or negates one number
/ // floating point division on two numbers
* // multiplies two numbers
\ // integer division on two numbers

Boolean Operators

! // negates a boolean value
== // compares two boolean values and returns true if they are equal
> // greater than
< // less than
>= // greater or equal to
<= // less than or equal to
and // true only if both values are true
or // false only if both values are false

Memory Operators

&identifier; // returns the memory address of the identifier
*address; // returns the value at the address in the memory

Note: *address cannot be used to modify memory values, for that, use the memset function.
