Syntax

// used in main
ret;
// used in a function
let f => (x) {
	ret value;
};

Usage

ret returns a given value to the function caller, or exits the main program.

The returned value will be evaluated beforehand and the local scope will be popped off the stack when ret is called.

let f => (x) {
	ret x + 1;
};
let y = 10;
f(y); // prints 11
