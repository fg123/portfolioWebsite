Syntax

// declare a variable
let indentifier;
// declare and initialize variable
let identifier = value;
// declare and initialize constant
let identifier := value;
// declare a array
let identifier[size];
// declare a function
let function_name => (param1, param2, ...) {
	function_body;
}

Usage

Let declares a identifier and optionally binds it to a value. The variable can either be bound as a constant value or a mutable value. Mutation is done with the set command.
