Syntax

input identifier;

Usage

input prompts for user input, then stores it in the given identifier.

If the user enters a value that can be evaluated as a number, the identifier will become a number type, any other input is a string type.

let name;
@"What is your name? ";
input name; // will be whatever the user typed in
@"Your name is: " + name;

What is your name? Wendy
Your name is: Wendy
