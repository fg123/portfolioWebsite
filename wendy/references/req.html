Syntax

req filename;

Usage

Wendy will look for the file in either the current folder of the file being executed or the wlib folder located in the installation of Wendy. (eg. if the wendy executable is built in the Wendy/bin/ folder, Wendy will look in the Wendy/wlib/ folder).

The found file's contents will simply be replaced in place of the req command and parsed as usual code.

req "wmath.w";
// functions defined in wmath.w can now be used, eg:
factorial(5);
